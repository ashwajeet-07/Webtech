Task 1: Data Binding

One Way Binding: Data flows in only one direction (Model → View or View → Model).

Subtypes:

1. Interpolation: Uses {{ '{' }}{{ '}' }} to display data from model to view.

2. Property Binding: Binds data from model to HTML element properties using syntax like [property]="value".

3. Attribute Binding: Binds data to HTML attributes such as colspan, aria-* when property binding is not applicable.

4. Event Binding: Handles user actions like clicks or input.

Two Way Binding: Data flows in both directions (Model ↔ View).

Type:

1. ngModel: Enables two-way data binding by combining property and event binding; commonly used in forms.
